# rcliment1987/skills

Skills personnels pour Claude Code — construits autour du stack BIMSmarter.

## Installation dans Claude Code

```
/plugin marketplace add rcliment1987/skills
```

Puis installe les skills :

```
/plugin install bimsmarter-prompt@rcliment1987-skills
/plugin install react-debug@rcliment1987-skills
/plugin install mistral-proxy@rcliment1987-skills
/plugin install bimsmarter-ui@rcliment1987-skills
/plugin install bimsmarter-n8n@rcliment1987-skills
/plugin install bimsmarter-firebase@rcliment1987-skills
/plugin install bimsmarter-security-audit@rcliment1987-skills
/plugin install systeme-monetisation-bimsmarter@rcliment1987-skills
/plugin install bimsmarter-gestion-entreprise@rcliment1987-skills
/plugin install bimsmarter-gestion-website@rcliment1987-skills
/plugin install bimsmarter-referencement-seo-geo@rcliment1987-skills
/plugin install bimsmarter-youtube@rcliment1987-skills
/plugin install prompt-ingenieur@rcliment1987-skills
/plugin install orchestrating-swarms@rcliment1987-skills
```

Redémarre la session. Les skills sont actifs dans tous tes projets.
