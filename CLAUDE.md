# CLAUDE.md — Repo skills-bimsmarter

Ce repo contient tous les skills Claude Code de BIMSmarter.
Le fichier `SKILLS_INDEX.md` à la racine est la source de vérité.

## Règles

- `SKILLS_INDEX.md` commande, la structure du repo obéit
- Ne jamais modifier le contenu des SKILL.md existants sans demande explicite
- Ne pas toucher au dossier `superpowers/` (plugin externe)
- Chaque skill = un dossier à la racine avec un SKILL.md dedans
- Les fichiers de référence vont dans `{dossier}/references/`
